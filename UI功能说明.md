# SKU拍照上传工具 - 前端功能说明

## 一、产品概述

这是一款移动端Web应用，用于仓库货盘SKU图片管理。用户通过手机浏览器访问，在同一WiFi下连接桌面端服务器，实现拍照上传功能。

**核心场景：** 从仓库系统同步暂存货盘SKU，为每个SKU的不同颜色拍照上传，自动归档命名。

---

## 二、功能模块

### 1. SKU列表模块

**数据来源：** 仓库系统暂存货盘（status='draft'）

**显示内容：**
- 货号（如 DX-A71-9965）
- 供应商代码（如 A71）
- 颜色信息（白色/深灰色）
- 码数段（S-XL / F）

**交互：**
- 点击选中，高亮显示
- 支持下拉刷新
- 显示总数统计

### 2. 颜色选择模块

**触发条件：** 选中SKU后显示

**显示内容：**
- 解析SKU的颜色字段，展示颜色按钮
- 支持多颜色：白色、深灰色、卡其色等
- 已上传图片的颜色标记为绿色

**交互：**
- 单选，点击选中
- 选中后展开拍照上传区域

### 3. 拍照上传模块

**功能：**
- 调用手机相机拍照（原图）
- 从相册选择照片
- 支持多选
- 显示选中照片预览

**上传：**
- 原图上传，不压缩
- 显示上传进度
- 上传成功后刷新已上传列表

**命名规则：**
```
{货号}-{颜色}-{序号}.{扩展名}
示例：DX-A71-9965-白色-1.jpg
```

### 4. 已上传图片模块

**显示内容：**
- 当前SKU已上传的所有图片缩略图
- 图片名称（文件名）
- 总数统计

**交互：**
- 点击可查看大图
- 支持删除（需二次确认）

## 三、设计风格要求

### 整体风格
- **现代简约**：干净、留白充足、无多余装饰
- **移动端优先**：大按钮、大字体、易点击
- **专业感**：适合工作场景，不花哨

### 配色方案
- **主色**：深蓝紫渐变（#667eea → #764ba2）
- **成功**：绿色（#4CAF50）
- **警告**：橙色（#FF9800）
- **背景**：浅灰（#f5f5f5）
- **卡片**：纯白，圆角12px

### 字体规范
- 标题：18px，加粗
- 正文：14-15px
- 辅助信息：12px
- 按钮：16px，加粗

### 交互规范
- 按钮点击有缩放反馈（scale 0.98）
- 选中状态有明显高亮
- Toast提示居中显示，2.5秒自动消失
- 加载状态有动画提示

### 布局规范
- 卡片间距：12px
- 内边距：16px
- 圆角：12px（卡片）、25px（按钮）
- 阴影：0 2px 8px rgba(0,0,0,0.08)

## 五、技术栈

- **前端**：原生HTML + CSS + JavaScript（无框架依赖）
- **后端**：Python Flask
- **通信**：RESTful API
- **图片传输**：FormData，原图上传
- **部署**：内嵌在Flask应用中，单文件运行

---

## 六、API接口

### 获取仓库暂存SKU
```
GET /api/v1/warehouse/drafts
Response: { success: true, data: [{id, 货号, 供应商代码, 颜色, 码数段, ...}] }
```

### 获取SKU已上传图片
```
GET /api/v1/skus/{sku_no}/images
Response: { success: true, data: [{file_name, file_path, url}] }
```

### 上传图片
```
POST /api/v1/skus/{sku_no}/upload
Content-Type: multipart/form-data
Body: color=白色, images=[file1, file2, ...]
Response: { success: true, added: ["DX-A71-9965-白色-1.jpg"], count: 1 }
```
